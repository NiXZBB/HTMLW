บทที่ 2 : คำสั่ง HTML เบื้องต้น

2.1 คำสั่ง HTML พื้นฐาน

คำสั่ง HTML เบื้องต้นที่มักจะใช้ในการสร้างหน้าเว็บมีดังนี้:

1. การใช้แท็กตัวอักษร (Text Formatting)

        
            <b>ข้อความตัวหนา</b>
            <i>ข้อความตัวเอียง</i>
            <u>ข้อความขีดเส้นใต้</u>
            <br>  
            <p>ข้อความในย่อหน้า</p>

ตัวอย่างการใช้งาน:

        
            <p>นี่คือ <b>ข้อความตัวหนา</b> และ <i>ข้อความตัวเอียง</i>.</p>

2. การเปลี่ยนแปลงรูปแบบตัวอักษร (Font Styling)

คุณสามารถกำหนดสี ขนาด และการจัดตำแหน่งตัวอักษรได้ดังนี้:

        
            <p style="color: blue; font-size: 20px;">
                ข้อความสีฟ้า ขนาด 20 พิกเซล
            </p>

            <p style="text-align: center; font-size: 24px;">
                ข้อความที่จัดให้อยู่ตรงกลาง
            </p>

ตัวอย่าง: ข้อความนี้มีสีแดงและจัดอยู่ตรงกลาง

3. การใช้ลิงก์ (Hyperlink)

        
            <a href="https://www.example.com">คลิกที่นี่เพื่อไปยัง Example.com</a>

ตัวอย่างการใช้งาน:

        
            <p>สามารถ <a href="https://www.example.com">คลิกที่นี่</a> เพื่อเยี่ยมชมเว็บไซต์ตัวอย่าง.</p>

4. การแทรกรูปภาพ (Image)

        
            <img src="image.jpg" alt="คำอธิบายรูปภาพ">

5. รายการไม่เรียงลำดับ (Unordered List)

        
            <ul>
                <li>รายการที่ 1</li>
                <li>รายการที่ 2</li>
                <li>รายการที่ 3</li>
            </ul>

2.2 การเชื่อมโยงระหว่างหน้า HTML

คำสั่งลิงก์สามารถใช้เพื่อเชื่อมโยงไปยังหน้าอื่นภายในเว็บไซต์ เช่น:

        
            <a href="บทที่ 1.html">ย้อนกลับไปบทที่ 1</a>

คุณสามารถใช้คำสั่งนี้เพื่อเชื่อมโยงหน้าต่าง ๆ ในเว็บไซต์ของคุณได้ง่าย ๆ.
