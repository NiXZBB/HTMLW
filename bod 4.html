บทที่ 4 : การสร้างตาราง

4.1 คำสั่งสร้างตาราง

ตารางใน HTML ใช้ในการจัดระเบียบข้อมูลในรูปแบบของแถวและคอลัมน์ โดยใช้แท็กต่าง ๆ ดังนี้:

        
            <table>  
                <tr>  
                    <th>หัวข้อ 1</th>  
                    <th>หัวข้อ 2</th>
                </tr>
                <tr>  
                    <td>ข้อมูล 1</td>  
                    <td>ข้อมูล 2</td>
                </tr>
            </table>

4.2 ตัวอย่างการสร้างตาราง

ตัวอย่างการสร้างตารางใน HTML:

        
            <table>
                <tr>
                    <th>ชื่อ</th>
                    <th>อายุ</th>
                </tr>
                <tr>
                    <td>สมชาย</td>
                    <td>25</td>
                </tr>
                <tr>
                    <td>สมหญิง</td>
                    <td>30</td>
                </tr>
            </table>

4.3 การจัดรูปแบบตาราง

สามารถใช้ CSS เพื่อปรับแต่งรูปแบบของตาราง เช่น สีพื้นหลัง ขนาดตัวอักษร และขอบของเซลล์:

        
            table {
                width: 100%;
                border-collapse: collapse;
            }
            th, td {
                border: 1px solid #ddd;
                padding: 8px;
                text-align: left;
            }
            th {
                background-color: #00796b;
                color: white;
            }
กลับไปหน้าแรก
