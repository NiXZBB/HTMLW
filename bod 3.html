บทที่ 3 : ใส่ภาพและลิงก์

3.1 การใส่ภาพใน HTML

การใส่ภาพใน HTML สามารถทำได้ง่าย ๆ โดยใช้แท็ก <img>. คำสั่งนี้มีแอตทริบิวต์ที่สำคัญดังนี้:

  • src: กำหนดที่อยู่ของภาพ เช่นไฟล์ในเครื่องหรือ URL ภายนอก
  • alt: กำหนดข้อความที่จะแสดงแทนหากไม่สามารถโหลดภาพได้
  • width และ height: กำหนดขนาดของภาพ (ไม่จำเป็นต้องระบุ)

ตัวอย่างการใส่ภาพใน HTML:

        
            <img src="image.jpg" alt="รูปภาพตัวอย่าง" width="500">

ในตัวอย่างนี้ จะมีการแสดงรูปภาพที่มีชื่อว่า image.jpg และกำหนดขนาดความกว้าง 500px.

3.2 การสร้างลิงก์ใน HTML

การสร้างลิงก์ใน HTML ทำได้โดยใช้แท็ก <a>. คำสั่งนี้มีแอตทริบิวต์ที่สำคัญคือ:

  • href: กำหนด URL หรือไฟล์ที่ต้องการเชื่อมโยง
  • target: กำหนดว่าลิงก์จะเปิดในหน้าต่างไหน (เช่น _blank สำหรับเปิดหน้าต่างใหม่)

ตัวอย่างการสร้างลิงก์ใน HTML:

        
            <a href="https://www.example.com" target="_blank">ไปที่ Example.com</a>

ในตัวอย่างนี้ ผู้ใช้สามารถคลิกเพื่อเปิดหน้าเว็บ https://www.example.com ในหน้าต่างใหม่ได้.

3.3 การทำลิงก์ในรูปภาพ

คุณสามารถสร้างลิงก์ที่คลิกได้จากภาพได้โดยการซ้อนแท็ก <a> ไว้รอบแท็ก <img>:

        
            <a href="https://www.example.com">
                <img src="image.jpg" alt="ลิงก์รูปภาพ" width="300">
            </a>

เมื่อผู้ใช้คลิกที่รูปภาพ ก็จะนำไปยัง URL ที่กำหนดไว้ใน href.
