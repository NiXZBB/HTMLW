บทที่ 1 : โครงสร้าง HTML

1.1 HTML คืออะไร?

HTML (Hypertext Markup Language) เป็นภาษาในการสร้างเว็บไซต์ โดยใช้โครงสร้างที่ทำให้เว็บเบราว์เซอร์สามารถแสดงผลข้อมูลได้อย่างถูกต้อง

1.2 โครงสร้างพื้นฐานของ HTML

โครงสร้างพื้นฐานของเอกสาร HTML มีลักษณะดังนี้:

        
            <!DOCTYPE html>
            <html lang="th">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>ชื่อเรื่อง</title>
            </head>
            <body>
                <h1>ยินดีต้อนรับสู่ HTML!</h1>
                <p>นี่คือเนื้อหาของคุณ</p>
            </body>
            </html>

1.3 อธิบายโครงสร้าง HTML

ในโครงสร้าง HTML จะมีส่วนประกอบหลัก ๆ ดังนี้:

  • DOCTYPE: ใช้เพื่อระบุประเภทของเอกสารว่าเป็น HTML5
  • html: แท็กหลักที่ครอบคลุมทุกส่วนของเอกสาร
  • head: ส่วนนี้มีข้อมูลเกี่ยวกับเอกสาร เช่น ชื่อเรื่อง (title)
  • body: ส่วนที่แสดงผลในเบราว์เซอร์ รวมถึงเนื้อหาต่าง ๆ เช่น ข้อความ รูปภาพ และลิงก์

1.4 ตัวอย่างการใช้ HTML

ตัวอย่างการสร้างหน้าเว็บด้วย HTML พื้นฐาน:

        
            <!DOCTYPE html>
            <html lang="th">
            <head>
                <meta charset="UTF-8">
                <title>หน้าเว็บตัวอย่าง</title>
            </head>
            <body>
                <h1>ยินดีต้อนรับสู่เว็บไซต์ของเรา</h1>
                <p>นี่คือเนื้อหาของเว็บไซต์</p>
            </body>
            </html>
